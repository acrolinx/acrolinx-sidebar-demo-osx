AcrolinxSidebarInterface Class Reference

Inherits from NSObject
Declared in AcrolinxSidebarInterface.h

Overview

AcrolinxSidebarInterface handles communication between the sidebar and the plugin. This class is instanciated and managed by AcrolinxSidebarController.
