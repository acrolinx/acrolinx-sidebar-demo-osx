AcrolinxSidebarDelegate Protocol Reference

Conforms to NSObject
Declared in AcrolinxSidebarInterface.h

Overview

A protocol used to communicate between the sidebars JS and the Acrolinx plugin.

This protocol contains methods for highlighting and replacing issues in the document. It is also needed to get the documents contents for checking and downloading attachments.
