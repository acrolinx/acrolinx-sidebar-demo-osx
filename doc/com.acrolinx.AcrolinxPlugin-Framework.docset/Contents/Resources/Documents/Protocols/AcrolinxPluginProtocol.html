AcrolinxPluginProtocol Protocol Reference

Conforms to NSObject
Declared in AcrolinxPluginProtocol.h

Overview

A Plugin protocol that defines methods to implement in an plugins principal class.

Plugins conforming to this protocol must know what types of files it can handle. The Acrolinx Application determines which of the plugins can handle a certain file extension and tells the corresponding instance to open the file.

– openFileAtPath: 

Opens a local file in the editor the plugin is written for.
- (NSString *)openFileAtPath:(NSString *)filePath

Parameters

filePath

A string representing a file to open.

Return Value

A path the file was actually written to, returns nil on error.

Discussion

It is possible that a plugin saves the file to a different location (or format). For this case the plugin should return the new file path so that the application can watchDog the correct file.
